# Incident Response Playbook

## Incident Lifecycle

1. Detect
2. Triage
3. Contain
4. Recover
5. Review

## First 15 Minutes Checklist

- Confirm incident scope and severity.
- Assign incident commander.
- Start timeline log.
- Apply immediate mitigation or rollback.

## Containment Actions

- Disable risky feature flags.
- Rate-limit abusive surfaces.
- Rotate compromised tokens/secrets.

## Recovery Validation

- Health checks green
- Error rate returns to budget
- Critical user flows pass smoke tests

## Postmortem Requirements

- Root cause
- User impact
- Detection/response gaps
- Preventive actions with owners and due dates
