# SLO Dashboard Contract

## Service Level Objectives

- Availability SLO: >= 99.5%
- Error-rate budget: < 1% requests
- Deployment success: 100% rollback-capable deployments

## Dashboard Panels

- Request volume by route
- 4xx/5xx error ratio
- p50/p95/p99 latency
- Consent analytics ingest outcomes (accepted/rejected)
- Webhook success/failure ratio

## Reporting Cadence

- Daily automated summary
- Weekly trend review
- Monthly SLO review and action list
