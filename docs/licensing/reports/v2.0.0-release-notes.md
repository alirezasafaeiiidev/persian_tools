- `<= v1.1.x`: MIT (historical releases unchanged)
- `>= v2.0.0`: Dual-licensed

1. `LICENSE` moved to dual-license selector.
2. `package.json#license` is now `SEE LICENSE IN LICENSE`.
3. `package.json#version` moved from `2.0.0-rc.1` to `2.0.0`.
4. Licensing governance finalized with `DCO + CLA` hybrid.
5. Release-prep automation and validation integrated in CI.
