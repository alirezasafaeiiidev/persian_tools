## Summary

From `v2.0.0`, Persian Tools uses a dual-license model:

- Non-Commercial license (free)
- Commercial license (paid)

## Version Boundary

- `<= v1.1.x`: MIT (unchanged historical releases)
- `>= v2.0.0`: Dual-licensed

## What Changed in RC

1. `LICENSE` switched to dual-license selector format.
2. `package.json#license` changed to `SEE LICENSE IN LICENSE`.
3. `package.json#version` set to `2.0.0-rc.1` for release-candidate validation.
4. Licensing docs/checklists/templates synchronized under `docs/licensing/`.

## Contributor Governance

External contributions follow `DCO + CLA` hybrid policy:

- `DCO.md`
- `docs/licensing/cla-individual.md`
- `docs/licensing/cla-corporate.md`
- `docs/licensing/cla-operations.md`

## Validation

- `pnpm licensing:validate`
- `pnpm ci:contracts`
- `pnpm ci:quick`
